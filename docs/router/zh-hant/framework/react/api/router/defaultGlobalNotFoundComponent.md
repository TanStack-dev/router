`DefaultGlobalNotFound` 元件是一個當沒有其他路由匹配且未提供 `notFoundComponent` 時，會在根路由上渲染「Not Found」的元件。

## DefaultGlobalNotFound 回傳內容

```tsx
<p>Not Found</p>
```
