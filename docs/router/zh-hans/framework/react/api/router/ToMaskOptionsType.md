`ToMaskOptions` 类型继承自 [`ToOptions`](./ToOptionsType.md) 类型，用于描述使用路由掩码 (route mask) 时可用的额外选项。

```tsx
type ToMaskOptions = ToOptions & {
  unmaskOnReload?: boolean
}
```
