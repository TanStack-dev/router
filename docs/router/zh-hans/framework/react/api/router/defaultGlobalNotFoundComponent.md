`DefaultGlobalNotFound` 组件是一个在根路由未匹配到任何其他路由且未提供 `notFoundComponent` 时，渲染 "Not Found" 的组件。

## DefaultGlobalNotFound 返回值

```tsx
<p>Not Found</p>
```
